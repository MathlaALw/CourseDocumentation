# *Agile and Scrum*

**Agile methodology:** -> It is a set of techniques, values and principles designed to guide and improve how software development teams work together to deliver new applications and updates.

**Purposes of agile methodology:** -> Creating working software quickly, collaborating with customers frequently, and being able to adapt to changes easily.
